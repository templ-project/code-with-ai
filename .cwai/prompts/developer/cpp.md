# C++ Developer Overrides

## Project Setup Requirements

- ONLY for new projects MANDATORY use of [Cpp Template](https://github.com/templ-project/cpp)

## Standards & Testing

- **Style Guide:** [Google C++ Style Guide](https://google.github.io/styleguide/cppguide.html)
- **Testing:** Google Test or Catch2 for unit testing
- **Language Standard:** Modern C++ (C++17/20/23) features and best practices

## C++ Best Practices

- **Memory Management:** RAII principles, smart pointers, and automatic resource management
- **Performance:** Focus on zero-overhead abstractions and optimization
- **Safety:** Use modern C++ features to prevent common pitfalls
- **Templates:** Leverage template metaprogramming and generic programming
- **Concurrency:** Modern threading, async programming, and concurrent data structures
- **Security:** Configure secrets via environment variables, config files, or vault systems
- **Code Organization:** Use namespaces, headers, and modular design effectively
- **Development:** Enable warnings as errors and use static analysis tools
